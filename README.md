# Parallel-Programming-Project

(The compile and run script only works in the linux environment.)

## Compile

* Serial code:

`./compile.sh serial`

* Parallel code:

`./compile.sh parallel`

## Run

`./run.sh [input size] [thread count]` (ex: `./run.sh 60 4`)
